This folder contains Catkin ROS nodes.

*IMPORTANT*

You have to initialize your Catkin workspace here by running from the src folder:

  $ cd src
  $ catkin_init_workspace
  $ cd ..

Now you can compile the workspace

  $ catkin_make

Set you environment variables

  $ source devel/setup.bash

Run roscore in one terminal

  $ roscore

Run hello in another terminal

  $ rosrun hello_coaches_developers hello

Read the message!
